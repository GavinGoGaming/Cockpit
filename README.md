# Cockpit
Cockpit is a Minecraft mod designed to make controlling the Elytra feel like a flight simulator. With smart guide lines, info meters, and renders, you'll accel at flying with or without rockets.

## Features
- Info display (Pitch, speed)
- Pitch guidelines
- Magnetic pitch (smooth snap pitch)
- Time since last firework rocket
- Fancy speed graph
- Stall and Dive warnings
- 3D Player render
